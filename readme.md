# Prime hashing algorithm

> ## [EN]

this hash takes advantage of the fact that every number has different distinct prime factors
by getting the sum of the prime factors' nth powers we can differentiate them from each other.

``example: f(30,2) -> 2^2 + 3^2 + 5^2 = 38``
> ## [TR]

bu hash her sayının farklı asal çarpanlara sahip olmasını kullanır
